Two Sentence Horror Gallery

They can also rate and comment on other users posts.
Anyone who visits the site can filter through latest, oldest and highest rated posts.
The authentication was built using Laravel Breezes built in authentication package.
This is my first backend project using a framework. Laravel was easy to use and well documented.
